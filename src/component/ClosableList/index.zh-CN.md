可关闭列表，支持选择列表中的元素，关闭列表中的元素。

## API
| 参数           | 说明                 | 类型                | 默认值         |
| :------------- | :------------------- | :------------------ | :------------- |
| selected   |  选中的项         | string || number |  - |
| data       |   列表中的数据   | function              | ​-             |
| onSelect   |  选中列表中一项的回调  | function  | -  |
| onClose    |  删除列表中的一项  | function  | -  |
| yTrue      | 列表头特殊的一项   | object | - |
